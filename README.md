# vue3-memo-app

プラクティス[create-vue](https://bootcamp.fjord.jp/practices/209)の提出物です

## プロジェクトのセットアップ

1. リポジトリをクローンします.

```sh
git clone https://github.com/Goto-star/vue3-memo-app
```

2. プロジェクトのディレクトリに移動します.

```sh
cd vue3-memo-app
```

3. 依存関係をインストールします。

```sh
npm install
```

4. 開発サーバーを起動し、アプリケーションをブラウザで確認します。

```sh
npm run dev
```
